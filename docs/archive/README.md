# Archived Documentation

This directory contains historical or superseded documentation.

These files are kept for context only. They are not the current product contract.

Typical reasons a document was archived:

- it described flags that do not exist in the current CLI
- it reflected early project planning rather than implemented behavior
- it captured a point-in-time status snapshot that is now stale
- it assumed Agentforce-only behavior before the provider abstraction was introduced

Archived docs should not be used as the primary source for:

- current command flags
- supported deployment workflows
- release readiness
- current AI provider behavior
